# Number-guessing-game

This is a simple **Number Guessing Game** built using **HTML, CSS, and JavaScript**.  

## How It Works
- The computer randomly chooses a number between 1 and 100.
- The player enters a guess and clicks the "Guess" button.
- The game tells the player if the guess is too high, too low, or correct.
- The total number of attempts is displayed.

## Features
- Interactive guessing game
- Tracks number of attempts
- Provides hints for each guess

## How to Play
1. Open `index.html` in your web browser.
2. Enter a number between 1 and 100.
3. Click the **Guess** button to see if your guess is correct.
4. Keep guessing until you find the correct number!
